# 📝 코딩테스트 오답노트

---

## 백준 1949 - 우수 마을

- **분류:** 트리 DP / 독립 집합
- **난이도:** 골드 2
- **링크:** https://www.acmicpc.net/problem/1949

---

### ❌ 틀린 코드

```java
// 플로이드-워셜 + DFS 시도 → 완전히 잘못된 접근
for(int k=1;k<n+1;k++) for(int i=1;i<n+1;i++) for(int j=1;j<n+1;j++) {
    vil[i][j] = Math.min(vil[i][j], vil[i][k]+vil[k][j]);
}

static void dfs(int v, int x) {
    for(int i=v+1;i<n+1;i++) {
        if(vil[v][i]==1||vil[v][i]==INF) {
            dfs(i, x);
            continue;
        }
        dfs(i, x+ans[i]);
    }
    if(v==n) {
        max = Math.max(x, max);
    }
}
```

---

### 🔍 틀린 이유

#### 1. 알고리즘 선택 자체가 잘못됨
플로이드-워셜은 **"모든 쌍 최단거리"** 를 구하는 알고리즘입니다.  
이 문제는 최단거리가 필요한 게 아니라 **"어떤 노드를 선택할지"** 를 결정하는 문제입니다.

#### 2. 독립 집합 조건을 체크하지 않음
```java
if(vil[v][i]==1||vil[v][i]==INF) {
    dfs(i, x);  // 인접하면 건너뜀 → 의도 자체가 반대
    continue;
}
dfs(i, x+ans[i]);  // 비인접이면 선택 → 완전히 반대 로직
```
인접한 노드를 **건너뛰는** 게 아니라, 인접한 노드를 **선택하지 않아야** 합니다.

#### 3. 플로이드-워셜은 이 문제에 구조적으로 부적합
| 문제 | 원인 |
|------|------|
| 메모리 초과 | n=10000 → 10000×10000 배열 = 약 4GB |
| 시간 초과 | O(n³) = 10¹² 연산 |
| 정확성 없음 | 최단거리로 독립 집합을 판단할 방법이 없음 |

#### 4. 문제 유형 파악 실패
> 트리 구조 + 노드 선택/비선택 조합 → **트리 DP**

### ✅ 정답 코드

```java
import java.util.*;
import java.io.*;

public class Main {
    static List<Integer>[] tree;
    static int[] population;
    static int[][] dp;
    static int n;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        n = Integer.parseInt(br.readLine());

        tree = new ArrayList[n + 1];
        population = new int[n + 1];
        dp = new int[n + 1][2];
        for (int i = 1; i <= n; i++) tree[i] = new ArrayList<>();

        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int i = 1; i <= n; i++) {
            population[i] = Integer.parseInt(st.nextToken());
        }

        for (int i = 0; i < n - 1; i++) {
            st = new StringTokenizer(br.readLine());
            int x = Integer.parseInt(st.nextToken());
            int y = Integer.parseInt(st.nextToken());
            tree[x].add(y);
            tree[y].add(x);
        }

        dfs(1, 0);
        System.out.println(Math.max(dp[1][0], dp[1][1]));
    }

    static void dfs(int v, int parent) {
        dp[v][0] = 0;
        dp[v][1] = population[v];

        for (int child : tree[v]) {
            if (child == parent) continue;
            dfs(child, v);

            dp[v][0] += Math.max(dp[child][0], dp[child][1]);
            dp[v][1] += dp[child][0];
        }
    }
}
```

---

### 💡 핵심 개념: 트리 DP

#### 상태 정의
```
dp[v][0] = v를 선택하지 않았을 때, v의 서브트리에서 가중치 합 최대
dp[v][1] = v를 선택했을 때, v의 서브트리에서 가중치 합 최대
```

#### 점화식
```
dp[v][0] += max(dp[child][0], dp[child][1])  // v 미선택 → 자식 자유롭게 선택
dp[v][1] += dp[child][0]                      // v 선택 → 자식은 반드시 미선택
```

#### 재귀가 끝나는 이유
트리는 사이클이 없기 때문에, `child == parent` 체크만으로 무한루프를 막을 수 있습니다.  
리프 노드에서는 이웃이 전부 `parent`뿐이므로 for문이 아무것도 실행하지 않고 자동으로 return됩니다.

```
dfs(1, 0)
  → dfs(2, 1)
      → child=1, parent=1 → continue → return  ← 리프에서 자동 종료
  → dfs(3, 1)
      → dfs(4, 3)
          → child=3, parent=3 → continue → return
```

#### 알고리즘 선택 기준
| 상황 | 알고리즘 |
|------|---------|
| 트리 + 선택/비선택 조합 | **트리 DP** ✅ |
| 모든 쌍 최단거리 (n ≤ 500) | 플로이드-워셜 |
| 단일 출발점 최단거리 | 다익스트라 |
