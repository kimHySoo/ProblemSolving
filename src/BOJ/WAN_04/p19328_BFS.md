# 📘 BOJ 19238 스타트 택시 오답노트

## 🧠 문제 핵심

- BFS로 가장 가까운 손님 찾기
- 거리 → 행 → 열 우선순위
- 손님 태우고 목적지까지 이동
- 연료:
  - 손님까지: `-dist`
  - 목적지까지: `+dist`

---

## ❌ 내가 틀린 이유

### 1. ❗ 상태 변경 순서 오류

#### 문제 코드
```java
k-=dist;
map[nx][ny]=0;
ok=pass;
if(!ok) break;
```

#### 문제점
- 손님을 못 찾았는데도 연료를 먼저 깎음
- map도 잘못 수정될 수 있음

#### ✅ 수정
```java
if(!pass) {
    ok = false;
    break;
}
k -= dist;
map[nx][ny] = 0;
```

---

### 2. ❗ 목적지 BFS에서 종료 안함

#### 문제 코드
```java
if(r==nx&&c==ny) {
    pass=true;
    dist = l;
}
```

#### 문제점
- 목적지를 찾고도 BFS 계속 진행

#### ✅ 수정
```java
if(r==nx&&c==ny) {
    pass = true;
    dist = l;
    break;
}
```

---

### 3. ❗ 손님 못 찾은 경우 처리 불명확

#### ✅ 해결 방식
```java
int dist = -1;
```

---

### 4. ❗ BFS 내부에서 r, c 덮어쓰기

#### 문제 코드
```java
r = x[0];
c = x[1];
```

#### ✅ 수정
```java
int cr = x[0];
int cc = x[1];
```

---

### 5. ❗ 손님 제거 확인을 size로 체크

#### ✅ 수정
```java
boolean found = false;
```

---

## ⚠️ 핵심 실수 패턴 정리

- 실패 조건은 항상 먼저 처리
- BFS는 찾으면 즉시 종료
- 상태 변수 분리 필수
- 못 찾는 경우는 명확히 표현 (dist = -1)
