# 📌 Manhattan Distance (맨해튼 거리) 정리

## 🧠 정의

두 점 A(x1, y1), B(x2, y2)의 맨해튼 거리:

\|x1 - x2\| + \|y1 - y2\|

👉 격자(grid)에서 상하좌우로만 이동할 때의 최단 거리

------------------------------------------------------------------------

## 📌 핵심 특징

### 1. 최단경로 = 거리

맨해튼 거리 = 실제 이동해야 하는 최소 칸 수

------------------------------------------------------------------------

### 2. 경로는 여러 개 존재

유클리드처럼 직선 1개가 아니라\
👉 여러 경로 존재 (조합 느낌)

------------------------------------------------------------------------

### 3. 거리 합 성질

임의의 점 P에 대해:

dist(A, P) + dist(P, B) ≥ dist(A, B)

👉 최단경로 위에 있으면 '=' 성립

------------------------------------------------------------------------

## 🔥 핵심 문제 유형

### 1. 두 점과 같은 거리인 점 찾기

조건:

\|x - xa\| + \|y - ya\| = \|x - xb\| + \|y - yb\|

------------------------------------------------------------------------

### 2. 해결 핵심

-   D = dist(A, B)
-   D가 홀수 → ❌ 불가능
-   D가 짝수 → ✅ 가능

------------------------------------------------------------------------

### 3. 이유

중간점 P가 존재하려면:

dist(A, P) = D / 2

👉 정수여야 하므로 D는 짝수

------------------------------------------------------------------------

## 🛠️ 풀이 전략

1.  두 점 사이 거리 계산
2.  절반 거리 계산
3.  한 방향(x → y 순)으로 이동

------------------------------------------------------------------------

## 💡 방향 처리 (signum 개념)

-   dx \> 0 → +1 방향
-   dx \< 0 → -1 방향

------------------------------------------------------------------------

## ⚠️ 실수 포인트

-   유클리드 중점 공식 사용 ❌
-   int overflow (좌표 1e9) ❌
-   입력 형식 실수 ❌

------------------------------------------------------------------------
