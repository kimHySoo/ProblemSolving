# 📘 음수 사이클 (Negative Cycle) 완전 정리

---

## 🔥 한 줄 핵심

> **음수 사이클 = 한 바퀴 돌 때마다 비용이 계속 줄어드는 사이클**

---

## 📌 정의

그래프에서 어떤 사이클이 있을 때:

```text
사이클 총 가중치 < 0
```

👉 이 경우를 **음수 사이클**이라고 한다.

---

## 📊 예시

```text
A → B (2)
B → C (3)
C → A (-10)
```

총합:

```text
2 + 3 - 10 = -5
```

👉 음수 사이클

---

## 💣 왜 문제가 되는가

음수 사이클이 있으면:

```text
한 번 돌면 -5
두 번 돌면 -10
세 번 돌면 -15
...
```

👉 계속 줄어듦 → **-∞ 가능**

---

## ❗ 핵심 결론

```text
음수 사이클이 있으면 최단거리는 존재하지 않는다
```

---

## 📌 중요한 오해

### ❌ dist 값이 음수면 음수 사이클?

```text
dist가 음수다 ≠ 음수 사이클
```

---

### 🔎 반례

```text
A → B (-5)
```

* dist[B] = -5 (음수)
* ❗ 사이클 없음

---

## 🔥 음수 사이클 판별 방법 (벨만포드)

### 핵심 아이디어

```text
정점 n개 → 최대 n-1개의 간선으로 최단거리 구성 가능
```

---

### 판별 기준

```text
n번째 반복에서도 값이 갱신되면 → 음수 사이클 존재
```

---

## 💻 코드 패턴

```java
for (int i = 1; i <= n; i++) {
    for (edge) {
        if (dist[to] > dist[from] + cost) {
            dist[to] = dist[from] + cost;

            if (i == n) {
                // 음수 사이클 존재
            }
        }
    }
}
```

---

## 📌 왜 n번째가 중요한가

* n-1번: 정상 최단거리 계산
* n번째: 더 줄어든다 → 이미 사이클 사용 중

👉 그 사이클이 계속 줄어든다 → 음수 사이클

---

## 🔥 모든 정점 고려 방법 (중요)

음수 사이클은 그래프 어디에 있을지 모름

---

### 방법 1️⃣ (추천)

```java
Arrays.fill(dist, 0);
```

👉 모든 정점을 시작점처럼 처리

👉 벨만포드 1번으로 전체 탐색 가능

---

### 방법 2️⃣ (비추천)

```text
모든 정점마다 벨만포드 수행
```

👉 시간초과 위험

---

## 📊 알고리즘별 가능 여부

| 알고리즘       | 음수 사이클 탐지 |
| ---------- | --------- |
| 다익스트라      | ❌         |
| Union-Find | ❌         |
| 벨만포드       | ✅         |
| SPFA       | ✅         |
| 플로이드 워셜    | ✅         |

---

## 💡 SPFA에서의 음수 사이클

```text
특정 정점이 n번 이상 큐에 들어가면 음수 사이클
```

---

## 📌 플로이드 워셜 판별

```java
if (dist[i][i] < 0) → 음수 사이클
```

---

## 🚨 실전에서 자주 틀리는 포인트

### ❌ dist 음수 체크

```java
if (dist[i] < 0) // 틀림
```

---

### ❌ 한 시작점만 탐색

```text
그래프 일부만 보고 판단 → 틀림
```

---

### ❌ 간선 덮어쓰기

```java
map[a][b] = w; // ❌
```

👉 반드시

```java
map[a][b] = Math.min(map[a][b], w);
```

---

## 🎯 문제 유형 정리

| 유형           | 해결 방법       |
| ------------ | ----------- |
| 최단거리 (음수 없음) | 다익스트라       |
| 최단거리 (음수 있음) | 벨만포드        |
| 음수 사이클 판별    | 벨만포드 / SPFA |
| 전체 쌍 최단거리    | 플로이드 워셜     |

---

## 🧠 최종 한 줄 정리

```text
음수 사이클 = n번째에도 계속 줄어드는 경로
```

👉 값이 음수인지가 아니라
👉 **계속 갱신되는지가 핵심**

---

## 🔥 실전 핵심 기억

```text
음수 사이클 판단 = “값”이 아니라 “갱신 여부”
```

---

👉 이거 완벽히 이해하면
**벨만포드 / SPFA / 플로이드 문제 거의 다 커버 가능**
