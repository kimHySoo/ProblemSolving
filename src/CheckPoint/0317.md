# 2D Prefix Sum (2차원 누적합)

직사각형 영역의 합을 빠르게 계산하기 위한 알고리즘.

------------------------------------------------------------------------

## 누적합 공식

ps\[i\]\[j\] = ps\[i-1\]\[j\] + ps\[i\]\[j-1\] - ps\[i-1\]\[j-1\] +
value

### 의미

-   위쪽 영역
-   왼쪽 영역
-   겹치는 영역 제거
-   현재 값 추가

------------------------------------------------------------------------

# long 연산

## 하나를 long으로 캐스팅
------------------------------------------------------------------------

```java
sum = (long)a + b;
```

👉 연산 자체가 long으로 수행됨 (안전)

## 누적 방식 사용 (추천)
------------------------------------------------------------------------

```java
sum += a;
sum += (long) a*b;
```

👉 `sum`이 long이므로 자동으로 long 연산 수행
