# 2D Prefix Sum (2차원 누적합)

직사각형 영역의 합을 빠르게 계산하기 위한 알고리즘.

------------------------------------------------------------------------

## 누적합 공식

ps\[i\]\[j\] = ps\[i-1\]\[j\] + ps\[i\]\[j-1\] - ps\[i-1\]\[j-1\] +
value

### 의미

-   위쪽 영역
-   왼쪽 영역
-   겹치는 영역 제거
-   현재 값 추가

------------------------------------------------------------------------

# long 연산

## 하나를 long으로 캐스팅
------------------------------------------------------------------------

```java
sum = (long)a + b;
```

👉 연산 자체가 long으로 수행됨 (안전)

## 누적 방식 사용 (추천)
------------------------------------------------------------------------

```java
sum += a;
sum += (long) a*b;
```

👉 `sum`이 long이므로 자동으로 long 연산 수행

------------------------------------------------------------------------

# 시간복잡도 관리

## 하나를 long으로 캐스팅
------------------------------------------------------------------------
```java
jewel.remove(i);
```

👉 중간 삭제 = **O(N)**  
👉 전체 성능 더 악화

### 같은 보석 반복 검사
- 매 가방마다 보석 리스트 처음부터 다시 탐색
- 이미 고려한 보석을 계속 다시 봄
- 처음 아이디어 부터 잘못됨

# ✅ 정석 풀이 (그리디 + 우선순위큐)

## 💡 핵심 아이디어
“현재 가방에 들어갈 수 있는 보석 중 가장 비싼 것 선택”

---
병목 원인

높이 후보마다 전체 배열을 다시 탐색함

같은 막대를 여러 높이 기준으로 반복 확인함

시간복잡도 O(n^2)

해결 방향

높이별 탐색이 아니라 막대별 탐색으로 바꾼다

각 막대가 자기 높이로 확장 가능한 최대 구간을 구한다

더 낮은 막대를 만나는 순간 넓이를 계산한다

스택으로 각 막대를 1번 push, 1번 pop 하여 O(n)에 해결

필요한 생각

“모든 높이를 다 볼까?” 대신

“이 막대는 어디까지 유지될까?”를 묻기

중복 계산을 줄이는 방향으로 기준을 바꾸기
